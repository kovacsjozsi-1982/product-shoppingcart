# Product Management App

This is a React + Vite project styled with TailwindCSS. The UI is generated directly from the Figma design: https://www.figma.com/design/Cep7R0EjWIdbO4GEzcAkti/Product-Management-Mockup---Codespring-LLM-Training?node-id=36-100&t=QWTnTgZnYBQqZ4FL-0

## Features
- Product list, details, create/edit/delete
- Responsive layout
- API integration via axios

## Project Structure
- `src/api` — API helpers
- `src/components` — reusable components
- `src/pages` — pages/screens from Figma frames
- `src/App.jsx` — main app entry

## Setup
```bash
npm install
npm run dev
```
